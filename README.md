Pokedot-Expansion

Un proyecto basado en Pokeemerald-Expansion y Pokemon Essentials, con la finalidad de crear una alternativa a la creación de juegos fan-made de pokémon, pero para distintas plataformas de forma nativa y optimizada.

Caracteristicas:
- Sistema de movimiento por casillas
- Sistema de Mapas (conexión entre mapas, cargado por chunks, colisiones mediante un tilemaplayer y custom data layer, bordes de mapa y atributos)
- Previsualización de mapas en el editor
- Sistema de Dia y Noche usando el reloj del sistema operativo y aplicando un filtro
- Sistema basico de pokemons, habilidades, movimientos e items (placeholdiers por ahora)
- Efectos de sonido y musica
- Sistema de caja de dialogos
- Sistema de guardado primitivo.
- Sistema de múltiples partidas (limite actual: 3, es expandible a más fácilmente)
